# django_vue

## Downloads

[Node](https://nodejs.org/en/download/)

[Python 3.10](https://www.python.org/downloads/)

## Installation

```bash
pip install -r requirements.txt
```

```bash
cd frontend
npm install
```

## Run Project

```bash
python manage.py runserver
```
to start django server

```bash
cd frontend
npm run
```
to start react server

In browser run
```bash
localhost:8080
```
